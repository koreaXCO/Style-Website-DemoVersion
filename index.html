JavaScript

기본문법

  1. 변수 (Variables)

    변수는 값을 저장하는데 사용됩니다. var, let, const 키워드로 변수를 선언할 수 있습니다.


    var x = 5; // 변수 선언
    let y = 10; // 블록 스코프 변수 선언
    const z = 15; // 상수 선언 (재할당 불가)

  2. 데이터 유형 (Data Types)

    자바스크립트의 주요 데이터 유형은 문자열, 숫자, 불리언, 배열, 객체 등이 있습니다.


    let name = "John"; // 문자열
    let age = 30; // 숫자
    let isStudent = true; // 불리언
    let numbers = [1, 2, 3, 4, 5]; // 배열
    let person = { name: "John", age: 30 }; // 객체

  3. 조건문 (Conditional Statements)

    조건문은 조건에 따라 코드 블록을 실행합니다.


    if (age >= 18) {
    console.log("You are an adult.");
    } else {
    console.log("You are a minor.");
    }

  4. 반복문 (Loops)

    반복문은 코드를 여러 번 실행합니다.


    for (let i = 0; i < 5; i++) {
    console.log(i);
    }
    let fruits = ["Apple", "Banana", "Orange"];
    for (let fruit of fruits) {
    console.log(fruit);
    }

  5. 함수 (Functions)

    함수는 코드를 재사용 가능한 단위로 묶어줍니다.


    function greet(name) {
    console.log("Hello, " + name + "!");
    }
    greet("John"); // 함수 호출

  6. 이벤트 처리 (Event Handling)

    이벤트 처리는 사용자 상호 작용에 대한 응답으로 코드를 실행합니다.


    document.getElementById("myButton").addEventListener("click", function() {
    console.log("Button clicked!");
    });

  7. 클래스 (Classes)

    클래스는 객체 지향 프로그래밍에서 사용되며, 객체 생성을 위한 템플릿 역할을 합니다.


    class Rectangle {
    constructor(width, height) {
    this.width = width;
    this.height = height;
    }
    getArea() {
    return this.width * this.height;
    }
    }
    let rect = new Rectangle(5, 10);
    console.log(rect.getArea());

실용/실무 문법

  1. 랜덤 숫자 생성하기

    1 이상 10 이하의 랜덤 정수 출력


    function getRandomNumber(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;
    }
    console.log(getRandomNumber(1, 10)); // 1 이상 10 이하의 랜덤 정수 출력

  2. 시간 형식 변환하기

    "?시 ?분 ?초"


    function formatTime(seconds) {
    let hours = Math.floor(seconds / 3600);
    let minutes = Math.floor((seconds % 3600) / 60);
    let remainingSeconds = seconds % 60;
    return `${hours}시간 ${minutes}분 ${remainingSeconds}초`;
    }
    console.log(formatTime(3665)); // 출력: "1시간 1분 5초"

  3. 이메일 유효성 검사

    이메일 양식 검사


    function validateEmail(email) {
    const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return regex.test(email);
    }
    console.log(validateEmail("example@example.com")); // 출력: true

  4. 날짜 형식 변환하기

    월(영) 일(영), 년도(영)


    function formatDate(dateString) {
    const options = { year: 'numeric', month: 'long', day: 'numeric' };
    return new Date(dateString).toLocaleDateString('en-US', options);
    }
    console.log(formatDate('2024-05-16')); // 출력: "May 16, 2024"

  5. 요소의 유무 확인하기

    출력: true / false


    function containsElement(arr, element) {
    return arr.includes(element);
    }
    let fruits = ['apple', 'banana', 'orange'];
    console.log(containsElement(fruits, 'banana')); // 출력: true

  6. 문자열에서 특정 패턴 검색하기

    출력: ['The', 'the']


    function searchPattern(str, pattern) {
    const regex = new RegExp(pattern, 'gi');
    return str.match(regex) || [];
    }
    let text = 'The quick brown fox jumps over the lazy dog';
    console.log(searchPattern(text, 'the')); // 출력: ['The', 'the']

  7. 클래스를 이용한 모듈화

    모듈화 작업


    class Calculator {
    add(a, b) {
    return a + b;
    }
    subtract(a, b) {
    return a - b;
    }
    multiply(a, b) {
    return a * b;
    }
    divide(a, b) {
    if (b === 0) {
    throw new Error('Division by zero');
    }
    return a / b;
    }
    }
    // 사용 예시
    let calc = new Calculator();
    console.log(calc.add(5, 3)); // 출력: 8

    마우스 클릭시


    document.getElementById('myButton').addEventListener('click', function(event) {
    console.log('Button clicked!');

  9. 마우스 이동 이벤트 처리

    마우스 이동시


    document.addEventListener('mousemove', function(event) {
    console.log(`Mouse position - X: ${event.clientX}, Y: ${event.clientY}`);
    });

  10. 마우스 오버/아웃 이벤트 처리

    마우스 IN/OUT 처리


    document.getElementById('myElement').addEventListener('mouseover', function(event) {
    console.log('Mouse over element');
    });
    document.getElementById('myElement').addEventListener('mouseout', function(event) {
    console.log('Mouse out of element');
    });

  11. 마우스 드래그 이벤트 처리

    마우스 드래그시


    let isDragging = false;
    document.getElementById('dragElement').addEventListener('mousedown', function(event) {
    isDragging = true;
    });
    document.addEventListener('mousemove', function(event) {
    if (isDragging) {
    console.log('Dragging...');
    }
    });
    document.addEventListener('mouseup', function(event) {
    isDragging = false;
    });

    마우스 휠 UP/DOWN

    });

    오른쪽 버튼 클릭방지
